# Salve o Curintia — Entrega 3 (JavaScript Avançado)

SPA com templates JS, roteamento via hash, validação de formulários (CPF, e-mail, telefone, CEP), máscaras leves e armazenamento local dos cadastros.

## Como rodar
Abra `index.html` no navegador (funciona direto em arquivo) ou publique no GitHub Pages.

## Estrutura
Veja `assets/js/` para módulos: `app.js`, `router.js`, `templates.js`, `formValidator.js`, `storage.js`, `utils.js`.

## Rotas (hash)
- `#/` — Home
- `#/projetos` — Lista e modal de doação (simulado)
- `#/cadastro` — Formulário com validação e LocalStorage
